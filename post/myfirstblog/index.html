Some Common Commands in Shell(My First Blog)

Dec 31, 2019
1 min read

Basic Commands

  • ls(list)

    • $ ls -l
    • $ ls -F

  • cp(copy)

    • $ cp file1 file2
    • $ cp file1 ... fileN dir

  • mv(move)

    • $ mv file1 file2
    • $ mv file1 ... fileN dir

  • touch

    • $ touch filename

  • rm

    • $ rm filename

  • echo

    • $ echo Hello again.

    Hello again.

Navigating Directories

  • cd(change directory)

    • $ cd dir

  • mkdir

    • $ mkdir dir

  • rmdir

    • $ rmdir dir
    • $ rmdir -rf dir

Intermediate Commands

  • grep

    • $ grep root /etc/passwd

  • less

  • pwd

  • diff

    • $ diff file1 file2

  • file

    • $ file filename
